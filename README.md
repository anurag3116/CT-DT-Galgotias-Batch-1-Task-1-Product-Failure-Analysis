Task 1 - Product Failure Analysis
Objective

To analyze the machine failure dataset, identify the factors responsible for product failure, and suggest possible solutions.

Dataset

Machine Failure Dataset from Kaggle.

Features Explored
Temperature
Leakage
Electricity
Speed Parameters
Other machine-related parameters
Analysis

The dataset was cleaned and explored to identify the conditions that may lead to machine failure.

Suggested Solutions
Regular maintenance
Monitoring of machine parameters
Stable operating conditions
Preventive actions to reduce failures
Files Included
[machine_failure_data excel.xlsx](https://github.com/user-attachments/files/28717232/machine_failure_data.excel.xlsx)

Conclusion

Proper monitoring and maintenance can improve machine reliability and reduce the chances of product failure.
